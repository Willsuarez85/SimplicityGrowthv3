# Templates

Reusable templates for consistent deliverables across all clients.

## Folders

| Folder | Purpose |
|--------|---------|
| `research/` | Brand audit templates, competitor analysis frameworks |
| `brand/` | Brand DNA templates, voice guidelines, avatar definitions |
| `content/` | Content calendar templates, pillar frameworks |
| `creative/` | Storyboard templates, script formats, brief structures |
| `delivery/` | Presentation templates, handoff checklists |

## Usage

Templates should be copied and customized for each client - never modify the original template files.
