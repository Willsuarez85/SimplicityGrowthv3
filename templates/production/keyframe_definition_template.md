## Generation Workflow

### Step 1: Generate Keyframe Images
1. Generate Keyframe 1 image using prompt
2. Generate remaining keyframes, referencing KF1 for consistency
3. Review for visual consistency
4. Regenerate inconsistent frames

### Step 2: Video Generation
**Option A: Image-to-Video**
- Use keyframes as start/end frames for AI video generation

**Option B: Video Prompts**
- Use keyframes as reference for pure video generation prompts

### Step 3: Assembly
- Combine generated segments
- Add transitions as specified
- Review for continuity
