## About This Asset

This moodboard is the visual reference for all IFIXX brand creative development. It synthesizes the complete brand identity system into a single visual reference.

## What's Included

The moodboard contains visual representations of:

1. **Logo & Branding**
   - Official IFIXX logo with correct colors
   - Logo usage examples and variations

2. **Color System**
   - Ifixx Gold (#e4bc5d)
   - Emergency Red (#ac1318)
   - Charcoal Black (#1c191a)
   - White (#ffffff)
   - Light Gray (#f5f5f5)

3. **Typography**
   - Inter font family (Bold, Semi-Bold, Medium, Regular)
   - Type scale and hierarchy examples
   - Readable text samples

4. **Photography Style**
   - Charlotte homes and neighborhoods
   - Remodeling work in progress
   - Professional, natural lighting
   - Owners visible when possible

5. **Iconography**
   - Line icons (2px stroke, 4px rounded corners)
   - Service category icons
   - Communication icons

6. **Brand Messaging**
   - Tagline: "Fast. Honest. Charlotte."
   - Key value propositions
   - Emergency vs. standard messaging tones

## How to Use This Moodboard

### For Creative Development
1. Reference this moodboard before creating any IFIXX graphic
2. Match colors exactly using the hex codes
3. Follow typography hierarchy shown
4. Maintain photography style consistency
5. Use iconography style for any custom icons needed

### For AI Image Generation
When using fal.ai or other AI tools, describe elements using this moodboard as reference:
- "Gold color (#e4bc5d) as shown in IFIXX moodboard"
- "Charcoal backgrounds like the IFIXX brand system"
- "Professional remodeling photography style, natural lighting, Charlotte homes"

### For Quality Control
Before finalizing any creative:
- [ ] Colors match moodboard hex codes exactly
- [ ] Typography follows Inter font family
- [ ] Photography style aligns with moodboard examples
- [ ] Icons match line style specifications
- [ ] Overall aesthetic feels consistent with moodboard
