## Template Categories

---

### 1. QUOTE POSTS

**Purpose:** Shareable wisdom, brand voice, engagement

#### Layout: Centered Quote
```
┌─────────────────────────────────────┐
│                                     │
│  [WATERMARK: "SIMPLICITY" at 3%]    │
│                                     │
│         "Quote text goes           │
│          here with impact           │
│          and clarity."              │
│                                     │
│         ─────────                   │
│         Attribution                 │
│                                     │
│         [LOGO]                      │
│                                     │
└─────────────────────────────────────┘
```

**Specifications:**
- Background: #0A0A0A or #000000
- Quote text: #FFFFFF, 32-40px, medium weight
- Attribution: #BDBDBD, 14px
- Optional: One turquoise accent word (use sparingly)
- Watermark: "SIMPLICITY" behind, 3% opacity

---

### 2. TIP/EDUCATIONAL POSTS

**Purpose:** Value delivery, expertise positioning

#### Layout A: Numbered Tip
```
┌─────────────────────────────────────┐
│                                     │
│  [LABEL: "GROWTH TIP"]              │
│                                     │
│        ┌─────┐                      │

│        └─────┘                      │
│                                     │
│     Tip headline goes here          │
│     with clear direction            │
│                                     │

│     that adds context.              │
│                                     │
│              [LOGO]                 │
│                                     │
└─────────────────────────────────────┘
```

**Specifications:**
- Background: #F7F7F7 (light) or #0A0A0A (dark)
- Label: #09B9B4, 12px uppercase, tracking 0.08em
- Number: 72px, bold, #0A0A0A or #FFFFFF
- Headline: 28px, semibold
- Body: 16px, regular

#### Layout B: Icon + Text
```
┌─────────────────────────────────────┐
│                                     │
│  [LABEL: "DID YOU KNOW?"]           │
│                                     │

│     actionable takeaway.            │
│                                     │
│              [LOGO]                 │
│                                     │
```

### Meta Ads: Lead Generation

#### Single Image Ad
```
┌─────────────────────────────────────┐
│                                     │
│         [MARBLE STATUE IMAGE]       │
│                                     │
└─────────────────────────────────────┘
│                                     │
│  HEADLINE: Value proposition        │
│                                     │
│  Body: Brief description that       │
│  explains the offer and creates     │
│  urgency to take action.            │
│                                     │
│  [■ Get Started Free]               │
│                                     │
└─────────────────────────────────────┘
```

**Specifications:**
- Image: Marble statue, dark background, 1080x1080
- Headline: 40 characters max
- Body: 125 characters max for optimal display
- CTA: Use platform's built-in buttons

#### Carousel Ad Structure
```
Slide 1: Hook/Problem
Slide 2: Agitation
Slide 3: Solution intro
Slide 4: Proof/Results
Slide 5: CTA
```

| Large Rectangle | 336 x 280px | High-impact |

## Visual Asset Integration

### When to Use Marble Imagery

| Post Type | Marble Image | Abstract/Minimal |
|-----------|--------------|------------------|
| Quote posts | ✓ Background | Alternative |
| Educational | ✓ Hero element | ✓ Clean focus |
| Testimonials | Optional | ✓ Preferred |
| Stats/Metrics | ✓ Dramatic | ✓ Clean data focus |
| Before/After | X | ✓ Required |
| Announcements | ✓ Impact | ✓ Clean alternative |
| Ads | ✓ Hero | Depends on objective |

### Marble Image Selection by Message

| Message Type | Recommended Image |
|--------------|-------------------|
| Leadership/Authority | THE COMMANDER, THE STRATEGIST |
| Innovation/Technology | THE PIONEER, THE VISION |
| Trust/Security | THE GUARDIAN |
| Results/Growth | THE FOUNDATION, THE MOMENTUM |
| Partnership | THE ALLIANCE |
| Precision/Quality | THE PRECISION |

## Color Application by Template

### Light Background Posts (#F7F7F7)
- Text: #0A0A0A
- Labels: #09B9B4
- Secondary text: #757575
- Borders: #EAEAEA

### Dark Background Posts (#0A0A0A or #000000)
- Text: #FFFFFF
- Labels: #09B9B4
- Secondary text: #BDBDBD
- Borders: #333333

### Turquoise Usage (5-8% Max)
- ONE label per post
- OR one accent word
- OR one CTA button (never all three)
- Never as full background
- Never as large text blocks

## Typography in Social

### Heading Hierarchy (Social)

| Element | Desktop Feed | Mobile/Stories |
|---------|--------------|----------------|
| Main Headline | 36-48px | 28-36px |
| Sub-headline | 24-28px | 20-24px |
| Body | 18-20px | 16-18px |
| Labels | 12-14px | 10-12px |
| Captions | 14-16px | 12-14px |

### Font Weights
- Headlines: Bold (700)
- Body: Regular (400)
- Labels: Semibold (600)
- CTAs: Semibold (600)

## Export Checklist

Before publishing any social asset:

- [ ] Correct dimensions for platform
- [ ] Text within safe zones
- [ ] Logo placement consistent
- [ ] Turquoise used sparingly (5-8% max)
- [ ] Background is pure black or appropriate gray
- [ ] Typography matches brand specs
- [ ] No more than ONE turquoise accent element
- [ ] File named correctly: `[client]_[platform]_[type]_[date].[ext]`
