## 1. El Concepto "Digital Marble"

### Filosofía Visual
Las estatuas greco-romanas representan la **sabiduría atemporal**, la estrategia y el pensamiento profundo. Al integrarlas con elementos tecnológicos, comunicamos que Simplicity Agency combina la claridad estratégica clásica con las herramientas de la era digital.

### Mensaje Visual
```
"No seguimos tendencias efímeras.
Aplicamos principios eternos con herramientas modernas."
```

## 2. Niveles de Intervención Tecnológica

### Nivel 1: Sutil (Background Tech)
**Uso:** Posts educativos, fondos de website

La estatua permanece intacta. La tecnología aparece solo en el entorno:
- Partículas de datos flotando sutilmente
- Grid digital muy tenue en el fondo
- Líneas de conexión tipo constelación
- Gradiente sutil de color tech (azul bajo)

**Ejemplo de prompt para AI:**
```
Greek marble bust of a philosopher, dramatic lighting,
dark background with very subtle floating data particles,
digital grid pattern barely visible, elegant minimalist,
high contrast, 8k quality
```

### Nivel 2: Moderado (Accessory Tech)
**Uso:** Posts de servicios, imágenes de features

La estatua porta o interactúa con UN elemento tecnológico:
- Auriculares inalámbricos premium
- Sosteniendo un smartphone elegante
- Tablet flotando cerca
- Smartwatch en muñeca de escultura

**Ejemplo de prompt para AI:**
```
Roman marble statue of a noble figure wearing
minimalist white wireless earbuds, contemplative pose,
soft dramatic lighting, clean dark gray background,
blend of classical and modern, photorealistic, 8k
```

### Nivel 3: Bold (Integrated Tech)
**Uso:** Hero images, campañas especiales

Integración más profunda pero elegante:
- Gafas AR translúcidas
- Holograma sutil cerca de la estatua
- Circuitos dorados en la toga/vestimenta
- Aura digital alrededor de la figura

**Ejemplo de prompt para AI:**
```
Greek goddess statue with translucent AR glasses
showing subtle holographic data, marble texture,
golden circuit patterns subtly visible on draped fabric,
dramatic chiaroscuro lighting, premium aesthetic,
dark background with soft golden glow, 8k photorealistic
```

## 3. Galería de Conceptos

### 3.1 Para Hero/Portadas

**Concepto: "The Strategist"**
```
Descripción: Busto de filósofo griego con auriculares
premium, mirando hacia arriba (visión), fondo oscuro
con partículas doradas sutiles.

Uso: Homepage hero, presentaciones corporativas
Orientación: Horizontal 16:9 o cuadrado
```

**Concepto: "The Builder"**
```
Descripción: Manos de escultura clásica sosteniendo
un tablet mostrando gráficos de crecimiento,
composición minimalista.

Uso: Página de servicios, ads
Orientación: Cuadrado o 4:5
```

**Concepto: "The Visionary"**
```
Descripción: Perfil de estatua con gafas AR
translúcidas, holograma de datos proyectándose,
fondo con gradiente oscuro a dorado.

Uso: Campañas de AI/tech, innovación
Orientación: Vertical 4:5 o 9:16 stories
```

### 3.2 Para Content/Posts

**Concepto: "Wisdom Speaks"**
```
Descripción: Primer plano de rostro de estatua
(boca y mentón), partículas digitales emanando
como si fueran palabras/ideas.

Uso: Posts de quotes, tips educativos
Orientación: Cuadrado 1:1
```

**Concepto: "Growth Foundation"**
```
Descripción: Columna griega parcial con gráfico
de crecimiento integrado en su base,
estilo arquitectónico meets data viz.

Uso: Posts de resultados, métricas
Orientación: Vertical 4:5
```

**Concepto: "Connected Minds"**
```
Descripción: Dos bustos clásicos de perfil
(uno mirando al otro), líneas de red neuronal
conectándolos sutilmente.

Uso: Colaboración, networking, community
Orientación: Horizontal 16:9
```

### 3.3 Para Elementos Decorativos

**Concepto: "Digital Fragment"**
```
Descripción: Fragmento de mano de estatua
(dedos, palma) con circuito dorado sutil visible,
fondo transparente/recortado.

Uso: Elemento decorativo en diseños, overlays
Formato: PNG con transparencia
```

**Concepto: "Marble Texture Tech"**
```
Descripción: Pattern repetible de textura de mármol
con líneas de circuito muy sutiles integradas.

Uso: Backgrounds de secciones, texturas
Formato: Tileable pattern
```

## 4. Paleta de Integración Tech

### Colores Tecnológicos Permitidos (uso mínimo)

| Color | Hex | Uso | Opacidad Max |
|-------|-----|-----|--------------|
| Cyan eléctrico | #00D4FF | Partículas, acentos | 30% |
| Azul profundo | #0A1628 | Fondos tech | 100% |
| Blanco data | #E8F4F8 | Líneas, grid | 20% |
| Dorado tech | #C9A227 | Circuitos, highlights | 60% |

### Regla de Aplicación
```
El dorado siempre predomina sobre el cyan/azul.
La tecnología se "baña" en los colores de marca,
no al revés.
```

## 5. Composición Visual

### Regla de Tercios Adaptada

```
┌───────┬───────┬───────┐
│       │       │       │
│  TECH │STATUE │ SPACE │
│ subtle│ focus │ breath│
├───────┼───────┼───────┤
│       │       │       │
│       │ MAIN  │       │
│       │SUBJECT│       │
├───────┼───────┼───────┤
│       │       │       │
│ SPACE │ TEXT  │ TECH  │
│       │ AREA  │ subtle│
└───────┴───────┴───────┘

- Estatua siempre en los 2/3 centrales
- Tech elements en las esquinas/bordes
- Espacio para texto en zona inferior o lateral
```

### Iluminación

**Estilo: Chiaroscuro Moderno**
```
- Fuente de luz principal: lateral superior (45°)
- Contraste alto pero no extremo
- Sombras profundas pero con detalle
- Highlights sutiles en puntos focales
- Si hay tech: luz secundaria sutil (color tech) desde atrás
```

## 8. Aplicaciones por Formato

### Instagram Feed (1080x1350)
```
Zona superior: Espacio o elemento decorativo
Zona media: Estatua protagonista
Zona inferior: Espacio para texto overlay
Tech elements: Corners o background sutil
```

### Instagram Stories (1080x1920)
```
Zona superior: Logo o headline
Zona media-superior: Estatua
Zona media-inferior: Espacio contenido
Zona inferior: CTA o swipe up
Tech elements: Laterales, partículas animadas
```

### Website Hero (1920x1080)
```
Lado izquierdo: Texto headline + CTA
Lado derecho: Estatua con tech elements
Background: Gradiente oscuro con grid sutil
Animación: Partículas flotando lentamente
```

### Thumbnail YouTube (1280x720)
```
Estatua: 60% del frame, lado derecho
Texto: Bold, lado izquierdo
Tech: Background particles, rim light
Contraste: Alto para visibilidad
```

## 9. Do's and Don'ts Específicos

### DO ✓
- Mantener la estatua como elemento noble y respetable
- Integrar tech de forma que "mejore" la estatua
- Usar iluminación cinematográfica
- Preservar la elegancia del mármol
- Crear contraste entre lo antiguo y moderno
- Mantener paleta de colores consistente

### DON'T ✗
- Hacer que la estatua parezca un robot/cyborg
- Cubrir el rostro completamente con tech
- Usar colores neón agresivos
- Crear composiciones recargadas
- Mezclar muchos elementos tech diferentes
- Perder la "humanidad" de la escultura
- Usar efectos glitch excesivos
- Hacer parecer que la estatua está "rota" por la tech

## 10. Checklist de Calidad

Antes de usar cualquier imagen del concepto "Digital Marble":

- [ ] La estatua es claramente el elemento protagonista
- [ ] Los elementos tech son complementarios, no dominantes
- [ ] La paleta de colores está dentro de la marca
- [ ] La iluminación es dramática pero elegante
- [ ] Hay suficiente espacio negativo
- [ ] La calidad técnica es alta (sin artefactos)
- [ ] El concepto comunica "premium boutique"
- [ ] La imagen funcionaría sin los elementos tech
- [ ] No hay elementos que distraigan
- [ ] El mensaje "Classical + Digital" es claro
