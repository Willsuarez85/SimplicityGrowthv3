## Visual Identity Direction

### Visual System: WHITEBOARD SYSTEMS™

**Core Concept:** Strategic Clarity, Engineered Growth

**Signature Formula:** Minimal + Whiteboard Cartoon + System Diagrams

**What It Communicates in 3 Seconds:**
1. "We build growth systems" — not random marketing
2. "We explain with clarity" — visual frameworks, funnels, flowcharts
3. "We're premium and clean" — minimal SaaS UI + subtle turquoise accents

---

### 4 Visual Layers

| Layer | Content |
|-------|---------|
| **A: Minimal Premium UI** | White/black, cards, soft shadows, Inter font |
| **B: Whiteboard Frameworks** | Funnels, pipelines, automation maps in black line art |
| **C: Simplicity People** | 6 character archetypes in consistent doodle style |
| **D: Watermarks & Grid** | SYSTEMS / GROWTH at 10-15% opacity background |

---

### Color Palette

| Color | Hex | Usage |
|-------|-----|-------|
| **White** | #FFFFFF | Primary backgrounds |
| **Off-white** | #F7F7F7 | Secondary backgrounds |
| **Black** | #000000 | Text, lines, illustrations |
| **Gray (text)** | #6B6B6B | Body copy |
| **Turquoise (accent)** | #09B9B4 | ONE highlight per composition |

**Turquoise Rule:** Maximum 5-8% of any composition. Only as underline, node highlight, or single accent.

---

### Design Principles

1. **Fewer elements, more intention** – Everything has purpose
2. **Always include a visual "map"** – Funnel, pipeline, diagram
3. **Turquoise only as highlight** – Never dominant
4. **Extreme consistency** – Same line style throughout
5. **Premium UI** – Whitespace, cards, soft borders

## Brand North Star Decision Framework

Every decision—strategic, creative, operational—must answer YES to:

1. **Does this help the partner grow their business?**
2. **Does this build a system (not just a tactic)?**
3. **Does this generate measurable outcomes?**
4. **Is this simple and clear to understand?**
5. **Does this reflect our partnership mentality?**

**If the answer to any question is NO, we don't do it.**
