## Prompt Philosophy

All prompts follow the **Whiteboard Systems™** visual identity:
- Black marker line art on off-white
- Professional business doodle style
- Clean, legible, consistent line weight
- Single turquoise accent (#09B9B4)
- Premium, modern, system-focused

---

## Master Style Keywords

Use these keywords consistently across all prompts:

```
Style keywords:
- "whiteboard cartoon style"
- "black marker line art"
- "professional business doodle"
- "clean and legible"
- "consistent line weight"
- "minimal shading"
- "off-white paper background"
- "lots of whitespace"
- "neat uppercase hand lettering"
- "premium growth agency"

Color keywords:
- "black marker outline"
- "off-white background (#F7F7F7)"
- "ONE subtle turquoise accent (#09B9B4)"
- "no bright colors"
- "no gradients"
- "no multicolor"

Avoid keywords:
- "no 3D"
- "no photorealism"
- "no clutter"
- "no childish illustration"
- "no messy scribbles"
- "not cartoon cute"
- "not meme style"
```

## Prompt Modifiers

Use these modifiers to adjust prompts:

### Aspect Ratios

```
Square (Instagram post): "1:1 aspect ratio, 1080x1080"
Story/Reels: "9:16 vertical format, 1080x1920"
LinkedIn: "horizontal 1200x628"
Facebook ad: "1:1 or 4:5 format"
Presentation: "16:9 widescreen format"
```

### Complexity Levels

```
Simple: "minimal elements, maximum 3-4 components"
Medium: "moderate detail, 5-6 components with connections"
Detailed: "complete system, multiple stages and branches"
```

### Emphasis Variations

```
Data-focused: "include specific numbers and metrics"
Process-focused: "emphasize flow and connections"
People-focused: "include character illustration"
Outcome-focused: "highlight results and benefits"
```

## Quality Checklist

Before using any generated asset, verify:

- [ ] Line weight is consistent throughout
- [ ] Only ONE turquoise accent is present
- [ ] Labels are legible and uppercase
- [ ] Style is professional (not childish or meme-like)
- [ ] Whitespace is abundant (40%+ empty)
- [ ] Diagram is clear and understandable
- [ ] Characters (if present) match Simplicity People style
- [ ] Background is off-white or white
- [ ] No gradients, 3D effects, or photorealism
- [ ] Asset feels premium and strategic

---

## Version Control

When generating new assets:

1. Reference this document for consistent styling
2. Use exact color codes (#09B9B4 for turquoise)
3. Maintain line weight consistency with existing assets
4. Follow character design rules for Simplicity People
5. Keep all frameworks in the same visual language
