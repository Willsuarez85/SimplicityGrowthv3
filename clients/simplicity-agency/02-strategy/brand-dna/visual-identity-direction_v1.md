## 1. Visión General

### Concepto Central
**"Classical Wisdom Meets Digital Innovation"**

Fusión de la elegancia atemporal de las estatuas greco-romanas con elementos tecnológicos contemporáneos, creando una identidad visual que comunica:
- Sabiduría estratégica (herencia clásica)
- Innovación tecnológica (era AI)
- Sofisticación premium (agencia boutique)

### Posicionamiento Visual
```
DE: Agencia de marketing genérica
A:  Boutique de crecimiento estratégico premium
```

## 3. Nuevos Elementos: Tecnología + Clásico

### 3.1 Concepto Visual Híbrido

**"Digital Marble"** - Estatuas clásicas intervenidas con:

| Elemento Clásico | Intervención Tecnológica |
|------------------|-------------------------|
| Busto romano | Auriculares/headset futuristas |
| Mano de escultura | Sosteniendo smartphone/tablet |
| Rostro clásico | Gafas de realidad aumentada |
| Toga drapeada | Patrones de circuitos sutiles |
| Pedestal | Base con luces LED/holográficas |
| Fondo neutro | Grid digital sutil, partículas |

### 3.2 Elementos Tecnológicos Permitidos
- Dispositivos móviles (smartphones, tablets)
- Auriculares inalámbricos premium
- Interfaces holográficas sutiles
- Partículas digitales / datos flotantes
- Líneas de conexión tipo red neuronal
- Gráficos de crecimiento minimalistas
- Iconografía de AI (neuronas, nodos)

### 3.3 Reglas de Integración
```
✓ La estatua siempre es protagonista
✓ Tecnología como accesorio, no invasión
✓ Colores tech en tonos fríos sutiles (cyan, azul eléctrico bajo)
✓ Nunca más de 2 elementos tech por composición
✓ Mantener elegancia y sofisticación
✗ Evitar aspecto "cyborg" o robótico
✗ No neon agresivo
✗ No glitch effects excesivos
```

## 4. Estilo Premium Boutique (Inspirado en Simplifiweb)

### 4.1 Principios de Diseño Limpio

| Principio | Aplicación |
|-----------|------------|
| **Whitespace generoso** | Mínimo 40% espacio en blanco por sección |
| **Tipografía jerárquica** | 3 niveles máximo: H1, Body, Caption |
| **Micro-interacciones** | Hover suaves, transiciones elegantes |
| **Iconografía consistente** | SVG monocromáticos, estilo lineal |
| **Secciones definidas** | Separadores sutiles (líneas, waves) |

### 4.2 Estructura de Página Web

```
┌─────────────────────────────────────────────┐
│  HERO                                        │
│  ├─ Video/imagen estatua + tech             │
│  ├─ Headline bold (max 8 palabras)          │
│  ├─ Subhead (1 línea)                       │
│  └─ CTA único prominente                    │
├─────────────────────────────────────────────┤
│  SERVICIOS                                   │
│  ├─ Cards con iconos SVG                    │
│  ├─ Título servicio + 3 palabras clave      │
│  ├─ Descripción breve (2 líneas)            │
│  └─ Hover: elevación + sombra sutil         │
├─────────────────────────────────────────────┤
│  FILOSOFÍA / DIFERENCIADOR                   │
│  ├─ Imagen estatua contemplativa            │
│  ├─ Quote o statement bold                  │
│  └─ Las 3 S's visualizadas elegantemente    │
├─────────────────────────────────────────────┤
│  TESTIMONIOS                                 │
│  ├─ Carousel minimalista                    │
│  ├─ Foto cliente + quote                    │
│  └─ Rating con estrellas doradas            │
├─────────────────────────────────────────────┤
│  CTA FINAL                                   │
│  ├─ Background estatua dramático            │
│  └─ Botón dorado prominente                 │
└─────────────────────────────────────────────┘
```

### 4.3 Componentes UI Premium

**Cards de Servicio:**
```css
/* Concepto visual */
- Border: none o 1px gris muy sutil
- Background: blanco o gris claro (#f8f8f8)
- Padding: generoso (32-48px)
- Shadow: sutil en hover (0 10px 30px rgba(0,0,0,0.08))
- Transición: 0.3s ease
- Icono: SVG monocromático centrado arriba
```

**Botones CTA:**
```css
/* Concepto visual */
- Background: dorado/champagne sólido
- Text: negro o blanco según contraste
- Padding: generoso horizontal
- Border-radius: 0 (rectangular) o muy sutil (4px)
- Hover: oscurecer 10%, escala 1.02
- Uppercase con letter-spacing
```

**Separadores de Sección:**
```css
/* Opciones */
- Wave SVG sutil (como Simplifiweb)
- Línea horizontal fina con gradiente
- Espacio en blanco generoso (80-120px)
```

## 5. Tipografía Premium

### Sistema Tipográfico Recomendado

| Uso | Fuente Sugerida | Peso | Características |
|-----|-----------------|------|-----------------|
| **Headlines** | Playfair Display / Cormorant | Bold | Serif elegante, clásica |
| **Subheads** | Montserrat / Raleway | SemiBold | Sans-serif geométrica |
| **Body** | Inter / Source Sans Pro | Regular | Sans-serif legible |
| **Captions** | Same as body | Light | Tamaño reducido |

### Jerarquía de Tamaños
```
H1: 48-64px (Desktop) / 32-40px (Mobile)
H2: 32-40px (Desktop) / 24-28px (Mobile)
H3: 24-28px (Desktop) / 20-22px (Mobile)
Body: 16-18px
Caption: 12-14px
```

### Reglas Tipográficas
- Letter-spacing en headlines: +0.02em
- Line-height body: 1.6-1.8
- Máximo 65-75 caracteres por línea
- Headlines en sentence case o uppercase selectivo

- **Grosor:** 1.5-2px consistente
- **Esquinas:** Redondeadas sutilmente

- **Tamaño:** 48-64px para features, 24px para UI

## 9. Lo Que NO Hacer

### Prohibiciones Visuales
```
✗ Fondos con muchos colores
✗ Gradientes neón o agresivos
✗ Stock photos genéricos de "business people"
✗ Iconos 3D o con sombras excesivas
✗ Efectos de texto (sombras, outlines, gradientes)
✗ Animaciones excesivas o que distraigan
✗ Layouts recargados o con mucho texto
✗ Fuentes decorativas o script
✗ Colores que no estén en la paleta
✗ Imágenes pixeladas o de baja calidad
```

### Errores Comunes a Evitar
1. Usar estatuas sin intervención tech (se ve "museo")
2. Abusar de elementos tech (se ve "startup genérica")
3. Mezclar muchos estilos fotográficos
4. Inconsistencia en tratamiento de color
5. Textos largos sin jerarquía visual
