## Overview

## Typography

### Font Characteristics

```
PRIMARY TYPOGRAPHY ATTRIBUTES:
• Sans-serif
• Geometric
• High legibility
• Clean weights (Regular / Medium / Bold)
• Modern, minimal aesthetic
```

### Recommended Font Families

| Priority | Font Family | Use Case |
|----------|-------------|----------|
| 1st | Inter | All-purpose, highly legible |
| 2nd | Space Grotesk | Headlines, impact |
| 3rd | DM Sans | Body text, readability |
| Alt | Geist | Tech-forward contexts |

### Typography Hierarchy

```
HEADLINE (Bold)
48-72px for hero content
32-40px for section headers

SUBHEAD (Medium)
24-28px for supporting headers
18-20px for card titles

BODY (Regular)
16-18px for paragraph text
14-16px for captions

MICRO (Regular/Medium)
12-14px for labels, metadata
```

## Graphic System: Whiteboard/Marker Style

### The Core Aesthetic

> "Real-time thinking sensation. Like watching someone work through an idea."

### Style Characteristics

```
┌─────────────────────────────────────────────────────────────────────────┐
│                     WHITEBOARD/MARKER SYSTEM                            │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│  STROKE STYLE                                                           │
│  • Marker-style strokes (2-4px weight)                                  │
│  • Slightly imperfect lines (human quality)                             │
│  • Hand-drawn feel without being childish                               │
│  • Confident, intentional marks                                         │
│                                                                          │
│  COMPOSITIONS                                                           │
│  • Simple geometric shapes (boxes, circles, arrows)                     │
│  • Flow diagrams and connections                                        │
│  • Annotations and labels                                               │
│  • Hierarchy through size and weight                                    │
│                                                                          │
│  FEELING                                                                │
│  • "I'm working through this with you"                                  │
│  • "This is how I think about it"                                       │
│  • "Simple enough to sketch on a whiteboard"                            │
│                                                                          │
└─────────────────────────────────────────────────────────────────────────┘
```

### Mode Selection

| Mode | Background | Stroke | Use Case |
|------|------------|--------|----------|
| Light Mode | White (#FFFFFF) | Black (#000000) | Default, tutorials, explainers |
| Dark Mode | Black (#000000) | White (#FFFFFF) | Impact, contrast, drama |

**Rule:** Never mix modes. Choose one per asset.

### Visual Elements

**Allowed:**
- Simple boxes and rectangles
- Circles and ellipses
- Arrows (straight and curved)
- Underlines and highlights
- Check marks and X marks
- Simple icons (line-drawn)
- Flow connections
- Hierarchy brackets

**Not Allowed:**
- Decorative illustrations
- Complex graphics
- Photorealistic elements
- 3D effects
- Shadows and gradients
- Colored fills
- Patterns or textures

## Asset Type Specifications

### YouTube Thumbnails

```
Dimensions: 1280 x 720px
Style: Typography-forward with whiteboard elements
Colors: Primarily monochromatic + sage green accent
Text: Bold, readable at small sizes
Elements: 1-2 visual elements maximum
```

**Prompt Example:**
```
YouTube thumbnail, clean dark background with subtle gradient,
bold white geometric sans-serif text "[TITLE]", single hand-drawn
marker-style diagram element in white, sage green (#87A96B) underline
accent, high contrast, professional minimal design, 16:9 aspect ratio
```

### Instagram Reels Covers

```
Dimensions: 1080 x 1920px (9:16)
Style: Vertical, bold typography
Colors: Monochromatic base + optional green accent
Text: Large, centered, maximum 3-4 words
```

**Prompt Example:**
```
Vertical Instagram cover, 9:16 aspect ratio, pure white background,
bold black geometric sans-serif text "[SHORT TEXT]", hand-drawn
marker underline in sage green (#87A96B), minimalist composition,
high contrast, clean professional aesthetic
```

### Instagram Carousels

```
Dimensions: 1080 x 1080px or 1080 x 1350px
Style: Educational, diagram-heavy
Colors: White background, black strokes
Slides: Consistent header placement
```

**Prompt Example (Carousel Slide):**
```
Educational carousel slide, square format, white background,
hand-drawn marker-style diagram showing [CONCEPT], black strokes,
simple geometric shapes, clean sans-serif labels, professional
whiteboard aesthetic, step [X] of framework, consistent header area
```

### Social Graphics

```
Dimensions: Platform-specific
Style: Quote cards, stat highlights
Colors: High contrast monochromatic
```

**Prompt Example:**
```
Social media quote card, minimalist design, white background,
black geometric sans-serif text reading "[QUOTE]", hand-drawn
quotation marks in marker style, sage green (#87A96B) accent line,
clean composition, professional editorial aesthetic
```

## Do's and Don'ts

### DO

| Category | Guidelines |
|----------|------------|
| **Colors** | Use monochromatic palette (black/white/gray) |
| **Colors** | Apply sage green sparingly for accents only |
| **Typography** | Use geometric sans-serif fonts |
| **Typography** | Maintain strong contrast and readability |
| **Graphics** | Keep diagrams simple and sketch-like |
| **Graphics** | Use marker-style strokes with human imperfections |
| **Photos** | Use black and white editorial style |
| **Photos** | Direct gaze, neutral expression, clean background |
| **Composition** | Embrace white space and minimalism |
| **Composition** | Create clear visual hierarchy |

### DON'T

| Category | Avoid |
|----------|-------|
| **Colors** | Never use colors beyond the defined palette |
| **Colors** | No gradients (except subtle photo backgrounds) |
| **Typography** | No decorative or script fonts |
| **Typography** | No low-contrast text combinations |
| **Graphics** | No complex illustrations or decorative elements |
| **Graphics** | No 3D effects, shadows, or glossy styles |
| **Photos** | No casual/lifestyle photography |
| **Photos** | No props, busy backgrounds, or context |
| **Composition** | No cluttered layouts |
| **Composition** | No mixing of light/dark modes in one asset |

## Quality Checklist

Before finalizing any visual asset, verify:

```
□ Monochromatic palette (black/white/gray base)
□ Sage green accent used sparingly (if at all)
□ High contrast maintained throughout
□ Typography is geometric sans-serif
□ Diagrams follow whiteboard/marker style
□ No decorative or unnecessary elements
□ Clear visual hierarchy established
□ Consistent with brand voice: "Clarity over personality"
□ Would pass the filter: "Does this help implement?"
```

## Imagen 4 Technical Notes

### Model Selection

| Model | Best For | When to Use |
|-------|----------|-------------|
| **Imagen 4** | Hero images, primary assets | Default choice |
| **Ideogram V3** | Text-heavy graphics, diagrams | When text accuracy critical |
| **FLUX Dev** | Quick iterations, testing | Rapid prototyping |

### Imagen 4 Strengths

- Excellent prompt adherence
- Strong typography rendering
- High-quality photorealistic output
- Good understanding of composition
- Reliable color accuracy

### Prompt Best Practices for Imagen 4

1. **Be Specific:** Include exact colors (#87A96B not "sage green")
2. **Describe Style:** "hand-drawn marker style" vs "sketch"
3. **Negative Context:** State what NOT to include
4. **Aspect Ratio:** Specify dimensions when relevant
5. **Quality Markers:** Use "professional", "high contrast", "clean"

### Common Prompt Modifiers

```
Quality: "professional", "high quality", "clean", "polished"
Style: "minimalist", "modern", "editorial", "whiteboard"
Mood: "confident", "clear", "intentional", "focused"
Technical: "high contrast", "sharp", "crisp lines"
```
