## 8. GENERATION WORKFLOW

### Step 1: Select Asset Type
Determine what you're creating (diagram, illustration, quote card, etc.)

### Step 2: Choose Base Prompt
Find the closest matching prompt from this library

### Step 3: Customize
- Replace placeholder text with specific content
- Adjust descriptions for your specific needs
- Maintain core style elements

### Step 4: Add Modifiers
Layer in style, mood, and technical modifiers as needed

### Step 5: Generate
- Use Ideogram V3 for text-heavy or diagram assets
- Use Imagen 4 for photographic or hero images
- Use FLUX Dev for fast iterations

### Step 6: Quality Test
Apply the 5-question test:
1. Does this explain a system?
2. Is it understandable without context?
3. Is there a human directing?
4. Is green signaling, not decorating?
5. Does it feel calm, not anxious?

### Step 7: Iterate
If any answer is "no," regenerate with adjusted prompt
